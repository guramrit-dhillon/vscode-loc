# Punjabi Language Pack for VS Code

Punjabi Language Pack provides localized UI for Visual Studio Code.

## ਪੰਜਾਬੀ ਭਾਸ਼ਾ ਪੈਕ

ਇਹ ਐਕਸਟੈਂਸ਼ਨ Visual Studio Code ਲਈ ਪੰਜਾਬੀ ਭਾਸ਼ਾ ਵਿੱਚ ਅਨੁਵਾਦਿਤ ਯੂਜ਼ਰ ਇੰਟਰਫੇਸ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

## Usage

Once installed, set `"locale": "pa"` in your `locale.json` to load the Punjabi Language Pack. To modify `locale.json`, press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS) to bring up the **Command Palette**, start typing "config" to filter the list, then select the **Configure Display Language** command.

## ਵਰਤੋਂ

ਇੰਸਟਾਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਪੰਜਾਬੀ ਭਾਸ਼ਾ ਪੈਕ ਨੂੰ ਲੋਡ ਕਰਨ ਲਈ ਆਪਣੀ `locale.json` ਫਾਈਲ ਵਿੱਚ `"locale": "pa"` ਸੈੱਟ ਕਰੋ।
