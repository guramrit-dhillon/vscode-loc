# Punjabi Language Pack for VS Code

ਇਹ ਐਕਸਟੈਂਸ਼ਨ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਦੇ ਇੰਟਰਫੇਸ ਨੂੰ ਪੰਜਾਬੀ ਭਾਸ਼ਾ ਵਿੱਚ ਅਨੁਵਾਦ ਕਰਦੀ ਹੈ।

This extension provides a Punjabi (ਪੰਜਾਬੀ) localization for the VS Code UI, written in the Gurmukhi script.

## Installation

1. Open VS Code
2. Open the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **"Punjabi Language Pack"**
4. Click **Install**

VS Code will prompt you to restart and switch to Punjabi. You can also switch manually:

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Run `Configure Display Language`
3. Select **pa (Punjabi)**
4. Restart VS Code

## What Gets Localized

This language pack translates the core VS Code UI, including:

- Menu bar and context menus
- Editor actions (split, close, save, format)
- File explorer (new file, new folder, delete, rename)
- Search and replace panel
- Source Control (Git) panel
- Debug toolbar and panels
- Extensions marketplace UI
- Settings editor
- Terminal panel
- Problems, Output panels
- Status bar items
- Notifications and dialogs
- Keyboard shortcut labels
- Accessibility strings

## Language Details

| Property | Value |
|----------|-------|
| Language ID | `pa` |
| Language Name | Punjabi |
| Localized Name | ਪੰਜਾਬੀ |
| Script | Gurmukhi |
| Region | Punjab (India / Pakistan) |
