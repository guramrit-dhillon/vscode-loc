# Hindi Language Pack for VS Code

यह एक्सटेंशन विजुअल स्टूडियो कोड के इंटरफेस को हिन्दी भाषा में अनुवाद करती है।

This extension provides a Hindi (हिन्दी) localization for the VS Code UI, written in the Devanagari script.

## Installation

1. Open VS Code
2. Open the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **"Hindi Language Pack"**
4. Click **Install**

VS Code will prompt you to restart and switch to Hindi. You can also switch manually:

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Run `Configure Display Language`
3. Select **hi (Hindi)**
4. Restart VS Code

## What Gets Localized

This language pack translates the core VS Code UI, including:

- Menu bar and context menus
- Editor actions (split, close, save, format)
- File explorer (new file, new folder, delete, rename)
- Search and replace panel
- Source Control (Git) panel
- Debug toolbar and panels
- Extensions marketplace UI
- Settings editor
- Terminal panel
- Problems, Output panels
- Status bar items
- Notifications and dialogs
- Keyboard shortcut labels
- Accessibility strings

## Language Details

| Property | Value |
|----------|-------|
| Language ID | `hi` |
| Language Name | Hindi |
| Localized Name | हिन्दी |
| Script | Devanagari |
| Region | India |
