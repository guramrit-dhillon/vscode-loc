# Hindi Language Pack for VS Code

Hindi Language Pack provides localized UI for Visual Studio Code.

## हिन्दी भाषा पैक

यह एक्सटेंशन Visual Studio Code के लिए हिन्दी भाषा में अनुवादित यूज़र इंटरफेस प्रदान करता है।

## Usage

Once installed, set `"locale": "hi"` in your `locale.json` to load the Hindi Language Pack. To modify `locale.json`, press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS) to bring up the **Command Palette**, start typing "config" to filter the list, then select the **Configure Display Language** command.

## उपयोग

इंस्टॉल करने के बाद, हिन्दी भाषा पैक को लोड करने के लिए अपनी `locale.json` फ़ाइल में `"locale": "hi"` सेट करें।
